# Studica Device VMX drivers
## Build and install
``` bash
make
make install
make clean
```

## Dependencies
* [VMX Hal CPP Library](https://github.com/Studica-Robotics/ubuntu/tree/main/hal/hal_cpp)

## Compile and run examples
``` bash
# build all examples
cd examples
make
# clean
make clean
```
``` bash
cd example_example
# build single example
make
# run example
./example_example
```
